# Editing a network

1. Go to **Networks**.

![](../img/i-net1-ua.svg)

**Using the context menu:**

- In the list of networks, find the network you want to edit.

- Expand the context menu of the network.

![](../img/i-net10-ua.svg)

- Select the **Edit** action.

![](../img/i-net11-ua.svg)

You can change the name and status of the network.
