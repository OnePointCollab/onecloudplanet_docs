# Configurations

When [creating an instance](/control-panel/cloud-platform/compute/instances/create-an-instance), you can select the required number of virtual processor cores (vCPU), random access memory (RAM) and the choice of local or block storage.

A fixed configuration is available from several lines at different locations, which differ in the ratio of resources and technical characteristics.

Once the instance has been created, you can [change the configuration](/control-panel/cloud-platform/compute/instances/type-instance).
