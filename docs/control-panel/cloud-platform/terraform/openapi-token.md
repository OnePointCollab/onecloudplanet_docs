# OpenAPI token


## Get OpenAPI token

1. Go to [OpenAPIToken](https://console.ocplanet.cloud/account-settings/api-access) page.

2. Click **Add Token**.

![](../img/tokens/1.png)

3. In the modal window, enter the token name.

![](../img/tokens/2.png)

![](../img/tokens/3.png)

4. Among the provided types, select **OpenAPI**.

![](../img/tokens/4.png)

5. Click **Add**.

![](../img/tokens/5.png)
