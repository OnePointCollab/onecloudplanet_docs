# OpenStack token


## Get OpenStack token

1. Go to [OpenStack token](https://console.ocplanet.cloud/account-settings/api-access) page.

2. Click **Add Token**.

![](../img/tokens/1.png)

3. In the modal window, enter the token name.

![](../img/tokens/2.png)

![](../img/tokens/3.png)

4. Among the provided types, select **OpenStack**.

![](../img/tokens/os.png)

5. Click **add**.

![](../img/tokens/5.png)
