# Редагування мережі

1. Перейдіть до **Мережі**.

![](../../img/i-net1-ua.svg)

**Через контекстне меню:**

- У списку мереж знайдіть потрібну мережу.

- Розгорніть контекстне меню мережи.

![](../../img/i-net10-ua.svg)

- Виберіть дію **Редагувати**.

![](../../img/i-net11-ua.svg)

Ви можете змінити ім'я та стан мережі.
